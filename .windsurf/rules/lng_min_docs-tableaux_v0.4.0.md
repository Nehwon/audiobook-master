# Documentation en Tableaux ([mineure])

La documentation doit commencer par des phrases complètes explicatives, puis utiliser des tableaux pour structurer les informations détaillées.

## Contexte d'utilisation
Cette règle s'applique à la création de fichiers de documentation technique. Elle améliore la lisibilité en combinant explications narratives avec présentations structurées des données.

**Version : 0.4.0** | **Catégorie : Langage/Français**
